# FEAT-ADMIN-002 Environments Management

This feature package covers the Environments Management capability in the Admin Portal.

Current target scope includes:

- environment and stage lifecycle visibility
- stage Azure resource configuration
- immediate orchestration across supported Azure action types
- recurring schedules
- notification recipient groups
- postponement of scheduled actions by authorized recipients
- activity and audit visibility
