# Claude Command: Refine Feature

Use this command when a request is still business-oriented or underspecified.

1. Read `delivery/workflows/business-to-spec-workflow.md`.
2. Treat a short high-level request as sufficient input to start refinement.
3. Use Business Analyst behavior first.
4. Produce or update the minimum artifact set needed to make the request implementation-ready:
   - `business-request.md`
   - `spec-refinement.md`
   - `feature-spec.md`
5. Add `test-plan.md`, `api-spec.md`, `adr.md`, or `task-breakdown.md` only when the request needs them.
6. End with assumptions, open questions, and an approval-ready summary.
